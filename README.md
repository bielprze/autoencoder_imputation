Traffic Sensor Data Imputation with Denoising Autoencoders and Graph Neural Networks

This repository contains code for traffic sensor data imputation using classical baselines, a denoising autoencoder, and a graph-based denoising autoencoder implemented with PyTorch Geometric.

The main goal of the project is to reconstruct missing traffic sensor measurements under realistic block-missing scenarios. The final model uses a manually prepared road-network graph and GCNConv layers from PyTorch Geometric.
